# 📌 Task 02: Stopwatch Web Application

## ⏱️ Stopwatch Web Application

This project is an interactive and user-friendly **Stopwatch Web Application** developed using HTML, CSS, and JavaScript. It allows users to accurately measure time intervals with essential stopwatch functionalities.

---

## 🚀 Features
- Start, Pause, and Reset functionality
- Lap time tracking and display
- Accurate time calculation using JavaScript
- Clean and modern user interface
- Responsive design for different screen sizes

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📚 What I Learned
- Handling time-based events using JavaScript
- Using `Date.now()` for accurate time tracking
- DOM manipulation and event handling
- Designing responsive and interactive UI
- Improving performance and efficiency in web applications
